Nabha Health Connect is a modern, AI-driven web application designed to analyze health symptoms. By describing their symptoms in natural language, users receive a preliminary analysis generated by advanced AI models. This empowers them to better understand their situation and seek appropriate medical care.

> [!WARNING]
> **Disclaimer:** This application is for informational purposes only and does not provide medical advice. The AI-generated analysis is not a diagnosis. **Always consult a qualified healthcare provider for any medical concerns or before making any health decisions.**

## âš™ï¸ How It Works

The application follows a simple yet powerful workflow:

1.  **Input:** The user enters their health symptoms into a text area.
2.  **API Call:** The app securely sends the symptoms to a backend AI service for processing.
3.  **AI Analysis:** The AI model analyzes the text and generates a structured, preliminary analysis based on the provided information.
4.  **Display:** The result is sent back to the app, formatted for readability, and displayed to the user with a clear disclaimer.

## ğŸš€ Getting Started

Follow these steps to get the project up and running on your local machine.

### Prerequisites

Make sure you have the following installed:
- [Node.js](https://nodejs.org/) (v18 or higher)
- [npm](https://www.npmjs.com/) or [yarn](https://yarnpkg.com/)

### Installation

1.  **Clone the repository:**
    ```bash
    git clone https://github.com/your-username/nabha-health-connect-app.git
    cd nabha-health-connect-app
    ```

2.  **Install dependencies:**
    ```bash
    npm install
    # or
    yarn install
    ```

3.  **Set up environment variables:**
    Create a `.env.local` file in the root of your project and add your AI service API key.
    ```
    # .env.local
    NEXT_PUBLIC_GEMINI_API_KEY="your_api_key_here"
    ```

4.  **Run the development server:**
    ```bash
    npm run dev
    # or
    yarn dev
    ```

5.  **Open your browser** and navigate to `http://localhost:3000`.

## ğŸ”§ Usage

1.  Navigate to the main page.

## ğŸ¨ Customization

-   **AI Service:** The AI backend can be easily swapped. Simply edit the service file in `services/` to point to a different API (e.g., OpenAI) or a custom model.
